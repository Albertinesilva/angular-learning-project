# ⚡ Curso Angular

⚡ Angular learning project featuring components, reactive forms, Angular Material, and SCSS styling. This project demonstrates practical Angular development, component-based architecture, and reactive programming concepts.

## 📂 Estrutura do Projeto

- **projetos-fundamentals**: exercícios básicos e fundamentos do Angular
- **projetos-components**: exemplos de criação e integração de componentes
- **projeto-reactive-forms**: aplicação com formulários reativos
- **resumo-angular-material**: estudo e aplicação de Angular Material
- **decorator**: exemplos de utilização de decorators no Angular

## 🛠️ Tecnologias Utilizadas

- Angular
- TypeScript
- HTML5
- SCSS
- Angular Material
- Reactive Forms

## 🚀 Funcionalidades

- Componentes reutilizáveis
- Formulários reativos com validações
- Layouts responsivos com Angular Material
- Demonstração de boas práticas de arquitetura de frontend

## 💻 Como Rodar

1. Clone o repositório:
2. 
```bash
git clone https://github.com/Albertinesilva/curso-angular.git
```

2. Instale as dependências:

```bash
npm install
```

3. Rode a aplicação:

```bash
ng serve
```

4. Acesse em `http://localhost:4200`

Esses projetos foram desenvolvidos como parte de um curso de Angular para aprendizado prático, mostrando boas práticas de frontend e integração de componentes, formulários e bibliotecas de UI.
